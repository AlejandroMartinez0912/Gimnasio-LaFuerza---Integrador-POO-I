# LaFuerza Gym — Sistema de gestión de gimnasio

Aplicación de escritorio para la administración integral de un gimnasio. Permite gestionar clientes, tutores, ejercicios, rutinas personalizadas y el seguimiento de entrenamientos.

Desarrollado como Trabajo Integrador de Programación Orientada a Objetos I (UNaM).

---

## Funcionalidades principales

- **Gestión de clientes** — alta, baja, modificación y búsqueda de socios
- **Gestión de tutores** — instructores asignados a clientes
- **Catálogo de ejercicios** — administración por grupo muscular
- **Rutinas personalizadas** — creación y asignación de rutinas con detalle de ejercicios
- **Seguimiento de entrenamientos** — registro histórico de sesiones por cliente
